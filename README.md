# Test task «Лабецкий Недвижимость»

## Installation

Using npm (port 5173)

```bash
  git clone https://github.com/NikSerdu/labeckiy_realestate
  cd labeckiy_realestate
  npm install
  npm run dev
```

Using docker compose (port 80)

```bash
  git clone https://github.com/NikSerdu/labeckiy_realestate
  cd labeckiy_realestate
  docker compose up -d
```

## How to use?

Find the src/components/Form/data/file.ts and add your field.
